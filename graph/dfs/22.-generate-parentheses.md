# 22. Generate Parentheses

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

For example, given n = 3, a solution set is:

```text
[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]
```

### Solution1: DFS

```python
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        res = []
        
        def dfs(l, r, path):
            if r == n:
                res.append(path)
                return
            if l < n:
                dfs(l + 1, r, path + '(')
            if r < l:
                dfs(l, r + 1, path + ')')
            
        dfs(0, 0, '')
        return res

```

### Solution2: Iterationï¼Œ stack

```python
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        res = []
        s = [("",0, 0)]
        while s:
            x, l, r = s.pop()
            if r == n:
                res.append(x)
            if l < n:
                s.append((x+"(", l+1, r))
            if r < l:
                s.append((x+")", l, r+1))
        return res
```

### Solution3: DP

```python
class Solution(object):
    def generateParenthesis(self, n):
        dp = [[] for i in range(n + 1)]
        dp[0].append('')
        for i in range(n + 1):
            for j in range(i):
                dp[i] += ['(' + x + ')' + y for x in dp[j] for y in dp[i - j - 1]]
        return dp[-1]
```
